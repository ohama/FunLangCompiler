Add the following helper function near the top of Elaboration.fs, AFTER the `open` declarations and type/helper definitions but BEFORE `testPattern`. Place it right before the `let rec testPattern` function:

```fsharp
/// Raise an error with source location in "file:line:col: message" format.
let private failWithSpan (span: Ast.Span) (fmt: Printf.StringFormat<'a, string>) : 'a =
    Printf.ksprintf (fun msg ->
        let loc = sprintf "%s:%d:%d" span.FileName span.StartLine span.StartColumn
        failwith (sprintf "%s: %s" loc msg)
    ) fmt
```

Key points:
- Uses `Printf.ksprintf` so existing format strings (%s, %A, %d) all work unchanged
- Return type is `'a` (bottom type via exception) so it works in any expression position
- Output format: `file:line:col: message` (colon-separated, matching standard compiler error format)
- Must be `private` since it's only used within this module

Verify it compiles: `dotnet build src/FunLangCompiler.Compiler/FunLangCompiler.Compiler.fsproj`
