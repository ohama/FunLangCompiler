
   - Track whether compilation succeeded (use a mutable `succeeded` flag, set to true only before `Ok ()` is returned — or better, move cleanup into each match arm)

   - On error: delete lowered and llFile, but KEEP mlirFile for debugging
   - Simplest approach: replace `finally` with explicit cleanup in each path. In each `Error` return, delete only `lowered` and `llFile`. In the final `Ok ()` path, delete all three. Remove the `finally` block.
