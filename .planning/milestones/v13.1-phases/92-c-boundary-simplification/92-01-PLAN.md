**C runtime changes (lang_runtime.c):**

1. `lang_string_sub(LangString* s, int64_t start, int64_t len)`: add `start = LANG_UNTAG_INT(start); len = LANG_UNTAG_INT(len);` as first lines.

2. `lang_string_slice(LangString* s, int64_t start, int64_t stop)`: add `start = LANG_UNTAG_INT(start); stop = LANG_UNTAG_INT(stop);` as first lines. NOTE: The -1 sentinel works correctly because LANG_UNTAG_INT(-1) = -1 >> 1 = -1 (arithmetic right shift preserves -1). Also, when no stop is provided, the compiler emits raw -1 constant. After this change the compiler should emit tagged -1 instead. Wait -- actually the compiler currently emits raw -1 for the "no stop" case and does NOT untag it (see line 3507: `None -> (stopVal, [])`). After Phase 92, the C function unconditionally untags stop. So raw -1 would become UNTAG(-1) = -1, which is correct. BUT the user-provided stop is currently tagged and gets untagged by the compiler. After Phase 92, the tagged stop goes directly to C which untags it. So both paths work: raw -1 sentinel untags to -1, tagged user stop untags correctly.

3. `lang_string_char_at(LangString* s, int64_t index)`: add `index = LANG_UNTAG_INT(index);` first, change return to `return LANG_TAG_INT((int64_t)(unsigned char)s->data[index]);`

4. `lang_string_indexof(LangString* s, LangString* sub)`: TAG the return value. Change `return 0;` to `return LANG_TAG_INT(0);`, change `return 0 - 1;` to `return LANG_TAG_INT(-1);`, change `return (int64_t)(found - s->data);` to `return LANG_TAG_INT((int64_t)(found - s->data));`

5. `lang_string_to_int(LangString* s)`: change return to `return LANG_TAG_INT((int64_t)strtol(s->data, NULL, 10));`

6. `lang_hashset_count(LangHashSet* hs)`: change to `return LANG_TAG_INT(hs->size);`

7. `lang_queue_count(LangQueue* q)`: change to `return LANG_TAG_INT(q->count);`

8. `lang_mlist_count(LangMutableList* ml)`: change to `return LANG_TAG_INT(ml->len);`

9. `lang_mlist_get(LangMutableList* ml, int64_t index)`: add `index = LANG_UNTAG_INT(index);` first. Return stays as-is (values stored tagged).

10. `lang_mlist_set(LangMutableList* ml, int64_t index, int64_t value)`: add `index = LANG_UNTAG_INT(index);` first.
