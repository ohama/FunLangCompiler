# Requirements Archive: v3.0 Language Completeness

**Status:** SHIPPED
**Completed:** 2026-03-26

## Operators

- [x] **OP-01**: Modulo 연산자 (`%`) → `arith.remsi`
- [x] **OP-02**: Char 리터럴 (`'A'`) → `int64` 변환 후 I64 처리
- [x] **OP-03**: PipeRight (`x |> f`) → `App(f, x)` 디슈가
- [x] **OP-04**: ComposeRight (`f >> g`) → `fun x -> g (f x)` 클로저 생성
- [x] **OP-05**: ComposeLeft (`f << g`) → `fun x -> f (g x)` 클로저 생성

## Pattern Matching Extensions

- [x] **PAT-06**: `when` 가드 — 패턴 매치 후 조건식 평가, false면 다음 arm
- [x] **PAT-07**: OrPat (`| P1 | P2 -> body`) — 여러 패턴이 같은 body 공유
- [x] **PAT-08**: ConstPat(CharConst) — char 리터럴 패턴 매칭 (`arith.cmpi eq`)

## Builtins

- [x] **BLT-01**: `failwith msg` — stderr에 메시지 출력 후 exit(1)
- [x] **BLT-02**: `string_sub s start len` — 부분 문자열 추출
- [x] **BLT-03**: `string_contains s sub` — 부분 문자열 포함 여부 (bool)
- [x] **BLT-04**: `string_to_int s` — 문자열 → 정수 변환 (`atoi`)
- [x] **BLT-05**: `char_to_int c` — char → int 변환
- [x] **BLT-06**: `int_to_char n` — int → char 변환
- [x] **BLT-07**: `print`/`println` 변수 문자열 지원 (현재 리터럴만)

## Range

- [x] **RNG-01**: `[start..stop]` — 정수 리스트 생성 (start부터 stop까지)
- [x] **RNG-02**: `[start..step..stop]` — step 간격 정수 리스트 생성
