## Milestone Summary

**Key Decisions:**
- Module system: compile-time AST flattening (no runtime module objects)
- prePassDecls recursive with shared exnCounter for unique exception tags
- Qualified names: two-arm desugar (FieldAccess for M.x/M.Ctor, App for M.f)
- File I/O: open-use-close internally (no exposed file handles, GC won't fclose)
- LangString_s named struct for cross-file forward declaration
- All file errors via lang_throw (catchable by try/with)

**Technical Debt:**
- Pre-existing MLIR domination bug with two named-ctor arms (workaround documented)
- FileImportDecl (multi-file import) deferred to v7.0
- get_args deferred (requires @main signature change)
