# Milestone v7.0: Imperative Syntax

**Status:** SHIPPED 2026-03-28
**Phases:** 28-29
**Total Plans:** 4

## Overview

Expression sequencing (;), array/hashtable indexing (.[]), if-then without else, and loop constructs (while/for) compile to native code.

## Phases

### Phase 28: Syntax Desugaring
- [x] 28-01: SEQ + ITE verification (zero code changes for SEQ; Tuple([]) unit fix + LetPat terminator fix for ITE)
- [x] 28-02: IndexGet/IndexSet runtime dispatch (LangHashtable tag field + lang_index_get/set)

### Phase 29: Loop Constructs
- [x] 29-01: WhileExpr elaboration (3-block header CFG + freeVars)
- [x] 29-02: ForExpr elaboration (block-arg loop counter + ascending/descending)

---

## Key Decisions
- SEQ/ITE: parser-level desugars, no new codegen needed
- IndexGet/IndexSet: runtime dispatch via tag=-1 sentinel in LangHashtable first field
- WhileExpr: 3-block CFG (header/body/exit), condition re-elaborated in body for mutable safety
- ForExpr: block argument %i:i64 for SSA-correct loop counter, immutable binding
