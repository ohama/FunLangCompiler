## v9.0 Requirements

### String Builtins
- [x] **STR-01**: `string_endswith` — 문자열 끝 검사 (C runtime lang_string_endswith)
- [x] **STR-02**: `string_startswith` — 문자열 시작 검사 (C runtime lang_string_startswith)
- [x] **STR-03**: `string_trim` — 문자열 공백 제거 (C runtime lang_string_trim)
- [x] **STR-04**: `string_concat_list` — 구분자 + 리스트 → 결합 문자열 (C runtime lang_string_concat_list)

### Char Builtins
- [x] **CHR-01**: `char_is_digit` — 숫자 판별
- [x] **CHR-02**: `char_to_upper` — 대문자 변환

- [x] **CHR-04**: `char_is_upper` — 대문자 판별

- [x] **CHR-06**: `char_to_lower` — 소문자 변환

### IO Builtins
- [x] **IO-01**: `eprintfn` — stderr 포맷 출력

### Hashtable Builtins
- [x] **HT-01**: `hashtable_trygetvalue` — (bool, value) 튜플 반환
- [x] **HT-02**: `hashtable_count` — 해시테이블 요소 수 반환

### List/Array Builtins
- [x] **LA-01**: `list_sort_by` — 비교 함수로 리스트 정렬
- [x] **LA-02**: `list_of_seq` — 컬렉션을 리스트로 변환
- [x] **LA-03**: `array_sort` — 배열 정렬
- [x] **LA-04**: `array_of_seq` — 컬렉션을 배열로 변환

### Collection Types
- [x] **COL-01**: StringBuilder — 생성, add, toString
- [x] **COL-02**: HashSet — 생성, add, contains, count
- [x] **COL-03**: Queue — 생성, enqueue, dequeue, count
- [x] **COL-04**: MutableList — 생성, add, index get/set, count

### Language Constructs
- [x] **LANG-01**: String slicing — `s.[start..end]`, `s.[start..]`
