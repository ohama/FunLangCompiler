  <action>
In lang_runtime.c, add these 4 functions (place them near the existing string functions like lang_string_contains):

1. `int64_t lang_string_endswith(LangString* s, LangString* suffix)` -- uses memcmp to check if s ends with suffix. Returns 1 if true, 0 if false. Handle suffix->length > s->length case (return 0).

2. `int64_t lang_string_startswith(LangString* s, LangString* prefix)` -- uses memcmp to check if s starts with prefix. Returns 1 if true, 0 if false. Handle prefix->length > s->length case (return 0).

3. `LangString* lang_string_trim(LangString* s)` -- strips leading and trailing whitespace (space, tab, newline, carriage return). Allocate new LangString via GC_malloc. Handle all-whitespace case (return empty string with length 0).

4. `LangString* lang_string_concat_list(LangString* sep, LangCons* list)` -- two-pass: first measure total length, then fill buffer. Cast each list element: `LangString* item = (LangString*)(uintptr_t)cur->head`. Add separator between items (not before first or after last). Handle NULL list (return empty string). Allocate via GC_malloc.

In lang_runtime.h, add declarations for all 4 functions after existing string function declarations.

See 31-RESEARCH.md for exact code examples for all 4 functions.
