In Elaboration.fs, add TWO pattern match arms for eprintfn near the existing eprint/eprintln arms:

1. Two-arg case: `eprintfn "%s" someString` -- desugar to eprintln:
```fsharp
| App (App (Var ("eprintfn", _), String (fmt, _), _), argExpr, _)
    when fmt = "%s" ->
    let (argVal, argOps) = elaborateExpr env argExpr
    let unitVal = { Name = freshName env; Type = I64 }
    let ops = [LlvmCallVoidOp("@lang_eprintln", [argVal]); ArithConstantOp(unitVal, 0L)]
    (unitVal, argOps @ ops)
```

2. Zero-arg case: `eprintfn "literal message"` -- desugar to eprintln of the literal:
```fsharp
| App (Var ("eprintfn", _), String (fmt, _), _) ->
    elaborateExpr env (App(Var("eprintln", Ast.unknownSpan), String(fmt, Ast.unknownSpan), Ast.unknownSpan))
```

IMPORTANT: Place the two-arg case BEFORE the one-arg case so pattern matching resolves correctly (more specific first).
