   - `typedef struct LangQueueNode { int64_t value; struct LangQueueNode* next; } LangQueueNode;`
   - `typedef struct { LangQueueNode* head; LangQueueNode* tail; int64_t count; } LangQueue;`
   - `LangQueue* lang_queue_create(void)` — GC_malloc struct, head=NULL, tail=NULL, count=0
   - `void lang_queue_enqueue(LangQueue* q, int64_t value)` — GC_malloc node, append at tail, count++
   - `int64_t lang_queue_dequeue(LangQueue* q)` — if empty call `lang_failwith("Queue.Dequeue: queue is empty")`, pop from head, count--, return value
   - `int64_t lang_queue_count(LangQueue* q)` — return q->count

2. **LangMutableList** struct and 5 functions (use exact code from 33-RESEARCH.md "COL-04: MutableList C runtime"):
   - `typedef struct { int64_t* data; int64_t len; int64_t cap; } LangMutableList;`
   - `LangMutableList* lang_mlist_create(void)` — GC_malloc struct, cap=8, GC_malloc data (8*8 bytes), len=0
   - `void lang_mlist_add(LangMutableList* ml, int64_t value)` — grow with GC_malloc+memcpy if full (NEVER realloc), append value, len++
   - `int64_t lang_mlist_get(LangMutableList* ml, int64_t index)` — bounds check, return ml->data[index]
   - `void lang_mlist_set(LangMutableList* ml, int64_t index, int64_t value)` — bounds check, set ml->data[index]
   - `int64_t lang_mlist_count(LangMutableList* ml)` — return ml->len
