  Add two new includes after existing `#include <stdlib.h>`:

  Add 8 new functions at the end of lang_runtime.c (before any closing guards, after the last Phase 26 function). Follow the EXACT code from 27-RESEARCH.md patterns 1-6:

  1. `LangCons* lang_read_lines(LangString* path)` — opens file, reads lines with fgets into 4096-byte buffer, strips \n and \r\n, builds forward LangCons* list using cursor technique (same as lang_range), each cell->head stores LangString* cast via (int64_t)(uintptr_t). Throws via lang_throw on fopen failure with "read_lines: file not found: " prefix.

  2. `void lang_write_lines(LangString* path, LangCons* lines)` — opens file with "w", iterates LangCons* list, casts each cell->head back to LangString* via (LangString*)(uintptr_t), writes with fwrite(s->data, 1, s->length, f) + fputc('\n', f). Silently returns on fopen failure (matches write_file behavior).

  3. `LangString* lang_stdin_read_line(void)` — reads from stdin using fgetc loop, terminates on '\n' OR EOF, uses dynamic GC_malloc buffer (start 256, double on overflow), returns LangString*.

  4. `LangString* lang_stdin_read_all(void)` — reads ALL of stdin using fgetc loop, terminates ONLY on EOF (not newline), uses dynamic GC_malloc buffer (start 1024, double on overflow), returns LangString*.

  5. `LangString* lang_get_env(LangString* varName)` — calls getenv(varName->data), throws via lang_throw if NULL with "get_env: variable '" + name + "' not set" message, otherwise copies result into GC_malloc'd LangString.

  6. `LangString* lang_get_cwd(void)` — calls getcwd(tmp, 4096) with stack buffer, throws via lang_throw on failure with "get_cwd: failed" message, copies result into GC_malloc'd LangString.

  7. `LangString* lang_path_combine(LangString* dir, LangString* file)` — concatenates dir + "/" + file (adds "/" separator only if dir doesn't already end with '/'), returns new GC_malloc'd LangString.

  8. `LangCons* lang_dir_files(LangString* path)` — opens directory with opendir, iterates with readdir, skips "." and "..", skips non-regular files (d_type != DT_REG && d_type != DT_UNKNOWN), builds full paths (path + "/" + d_name) as LangString* in LangCons* forward list using cursor technique, throws on opendir failure with "dir_files: directory not found: " prefix, calls closedir.

  CRITICAL: All allocations use GC_malloc, never malloc/realloc/free. All error cases use lang_throw (catchable), not lang_failwith (exits). All pointer-to-int casts go through (uintptr_t) intermediate.
  </action>
