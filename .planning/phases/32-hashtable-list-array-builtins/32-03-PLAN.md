1. **static comparator for qsort** (place before lang_array_sort):
```c
static int lang_compare_i64(const void* a, const void* b) {
    int64_t x = *(const int64_t*)a;
    int64_t y = *(const int64_t*)b;
    if (x < y) return -1;
    if (x > y) return  1;
    return 0;
}
```

2. **lang_array_sort** -- in-place sort using qsort, void return:
```c
void lang_array_sort(int64_t* arr) {
    int64_t n = arr[0];
    if (n <= 1) return;
    qsort(&arr[1], (size_t)n, sizeof(int64_t), lang_compare_i64);
}
```
Array layout: arr[0] = length, arr[1..n] = elements. Sort starts at &arr[1].

3. **lang_array_of_seq** -- delegates to existing lang_array_of_list (Phase 32 tests list-input only):
```c
int64_t* lang_array_of_seq(void* collection) {
    /* Phase 32: treat input as LangCons* (list).
     * Future phases add dispatch for HashSet/Queue/MutableList inputs. */
    return lang_array_of_list((LangCons*)collection);
}
```

In lang_runtime.h, add declarations:
```c

```

NOTE: lang_compare_i64 is static (not in header). lang_array_of_list already exists -- no new declaration needed for it.
