  Wrappers to add:
  - `lang_sprintf_1i(char* fmt, int64_t a)` -- handles %d, %x, %02x, %c (cast a to (long))
  - `lang_sprintf_1s(char* fmt, char* a)` -- handles %s
  - `lang_sprintf_2ii(char* fmt, int64_t a, int64_t b)` -- two int args
  - `lang_sprintf_2si(char* fmt, char* a, int64_t b)` -- string + int
  - `lang_sprintf_2is(char* fmt, int64_t a, char* b)` -- int + string
  - `lang_sprintf_2ss(char* fmt, char* a, char* b)` -- two strings

  All return `LangString*` allocated via GC_malloc. Include `if (len < 0) len = 0;` safety check after first snprintf call. See 39-RESEARCH.md "Code Examples" section for exact implementations.

  **Step 1 -- sprintf pattern match arms in elaborateExpr:**

  CRITICAL ORDERING: 2-arg arms MUST come BEFORE 1-arg arms. If 1-arg comes first, `sprintf "%s=%d" name value` will partially match and produce wrong results (see Pitfall 1 in research).

  Add these arms to the `elaborateExpr` match expression (find existing builtin arms like `eprintfn` as anchor):

  **2-arg sprintf arm (FIRST):**
  ```fsharp
  | App (App (App (Var ("sprintf", _), String (fmt, _), _), arg1Expr, _), arg2Expr, _)
      when (let specs = fmtSpecTypes fmt in specs.Length = 2) ->
  ```
  - Call `fmtSpecTypes fmt` to get the 2-element spec list
  - Use `addStringGlobal env fmt` + `LlvmAddressOfOp` for format string (NOT elaborateExpr -- avoids unnecessary GEP+Load per Pitfall 2)
  - Match on spec combinations: [IntSpec;IntSpec] -> @lang_sprintf_2ii, [StrSpec;IntSpec] -> @lang_sprintf_2si, [IntSpec;StrSpec] -> @lang_sprintf_2is, [StrSpec;StrSpec] -> @lang_sprintf_2ss
  - For string args: extract char* data via `LlvmGEPStructOp(ptr, argPtr, 1)` + `LlvmLoadOp`
  - For int args: use `coerceToI64Arg`
  - See 39-RESEARCH.md "sprintf 2-arg arm" code example for exact implementation

  **1-arg sprintf int arm (SECOND):**
  ```fsharp
  | App (App (Var ("sprintf", _), String (fmt, _), _), argExpr, _)
      when (let specs = fmtSpecTypes fmt in specs.Length = 1 && specs.[0] = IntSpec) ->
  ```
  - `addStringGlobal env fmt` + `LlvmAddressOfOp` for format
  - `coerceToI64Arg` for argument
  - Call `@lang_sprintf_1i`

  **1-arg sprintf string arm (THIRD):**
  ```fsharp
  | App (App (Var ("sprintf", _), String (fmt, _), _), argExpr, _)
      when (let specs = fmtSpecTypes fmt in specs.Length = 1 && specs.[0] = StrSpec) ->
  ```
  - `addStringGlobal env fmt` + `LlvmAddressOfOp` for format
  - `coerceToPtrArg` for argument, then GEP+Load to extract char* data
  - Call `@lang_sprintf_1s`

  **Step 2 -- printfn desugar arms (BEFORE sprintf arms):**

  printfn desugars to `println(sprintf ...)` at elaboration time. Add these arms:

  **printfn with no format args (just literal string):**
  ```fsharp
  | App (Var ("printfn", _), String (fmt, _), _) ->
      let s = Ast.unknownSpan
      elaborateExpr env (App(Var("println", s), String(fmt, s), s))
  ```

  **printfn 2-arg (BEFORE 1-arg):**
  ```fsharp
  | App (App (App (Var ("printfn", _), String (fmt, _), _), arg1Expr, _), arg2Expr, _) ->
      let s = Ast.unknownSpan
      let sprintfExpr = App(App(App(Var("sprintf", s), String(fmt, s), s), arg1Expr, s), arg2Expr, s)
      elaborateExpr env (App(Var("println", s), sprintfExpr, s))
  ```

  **printfn 1-arg:**
  ```fsharp
  | App (App (Var ("printfn", _), String (fmt, _), _), argExpr, _) ->
      let s = Ast.unknownSpan
      let sprintfExpr = App(App(Var("sprintf", s), String(fmt, s), s), argExpr, s)
      elaborateExpr env (App(Var("println", s), sprintfExpr, s))
  ```

  ORDERING within elaborateExpr: printfn 2-arg -> printfn 1-arg -> printfn 0-arg -> sprintf 2-arg -> sprintf 1-arg-int -> sprintf 1-arg-str. All BEFORE the general App catch-all arm.
