Modify the `main` function to route based on the first non-flag argument:

**CLI routing logic** (after parseArgs):
```
match remaining with
| "build" :: rest -> handleBuild optLevel rest
| "test" :: rest  -> handleTest optLevel rest
| inputPath :: _  when inputPath.EndsWith(".fun") -> // existing single-file mode
| [] -> // usage message (updated to include build/test)
| other :: _ -> eprintfn "Error: unknown command or file: %s" other; 1
```

**handleBuild (optLevel: int) (args: string list) : int**
1. Call `ProjectFile.findFunProj()`. If `None`, print `"Error: funproj.toml not found (searched from current directory upward)"` to stderr and return 1 (ERR-01).
2. Read and parse: `let content = File.ReadAllText(path)` then `let config = ProjectFile.parseFunProj content (Path.GetDirectoryName(Path.GetFullPath(path)))`.
3. Determine targets:
   - If `args` is empty: use all `config.Executables` (CLI-01).
   - If `args` is `[name]`: find target where `t.Name = name` in `config.Executables`. If not found, print `"Error: no executable target named '%s'" name` to stderr and return 1 (ERR-03).
4. Create `build/` directory: `Directory.CreateDirectory(Path.Combine(config.ProjectDir, "build"))` (OUT-01).
5. For each target:
   - Resolve source: `let srcPath = ProjectFile.resolveTarget config target`. If file doesn't exist, print `"Error: target file not found: %s" srcPath` to stderr and return 1 (ERR-02).
   - Compute output: `Path.Combine(config.ProjectDir, "build", target.Name)`.
   - Compute prelude: `config.PreludePath |> Option.map (fun p -> Path.GetFullPath(Path.Combine(config.ProjectDir, p)))` (OUT-03).
   - Start timer: `let sw = System.Diagnostics.Stopwatch.StartNew()`.
   - Call `compileFile preludeDir srcPath outputPath optLevel`.
   - If success (returns 0), print `"OK: %s -> build/%s (%.1fs)" target.Name target.Name sw.Elapsed.TotalSeconds` (OUT-02).
   - If failure (returns nonzero), return that exit code immediately.
6. Return 0 if all targets compiled successfully.

**handleTest (optLevel: int) (args: string list) : int**
1. Same funproj.toml discovery as handleBuild (ERR-01).
2. Same target resolution but using `config.Tests` instead of `config.Executables`.
3. If `args` is `[name]`: find in `config.Tests`. If not found, print `"Error: no test target named '%s'" name` (ERR-03).
4. For each test target:
   - Resolve source path. Check exists (ERR-02).
   - Output path: temp file or `build/<name>` (use build/ to keep it consistent).
   - Compile via `compileFile`.
   - If compilation fails, report and continue to next test (don't abort all tests on first failure).
   - If compilation succeeds, run the binary: `Process.Start` with `ProcessStartInfo(outputPath, UseShellExecute=false)`, wait for exit.
   - If exit code = 0: print `"PASS: %s (%.1fs)" target.Name elapsed`.
   - If exit code != 0: print `"FAIL: %s (exit %d, %.1fs)" target.Name exitCode elapsed`.
5. Print summary: `"%d/%d tests passed" passCount totalCount`.
6. Return 0 if all passed, 1 if any failed.

