**lang_runtime.c:**
- Add unified hash function that checks `key & 1`:
  ```c
  static uint64_t lang_ht_hash_key(int64_t key) {
      if (key & 1) {
          // Tagged int: use murmurhash on untagged value
          uint64_t h = (uint64_t)LANG_UNTAG_INT(key);
          h ^= h >> 33; h *= 0xff51afd7ed558ccdULL; h ^= h >> 33;
          h *= 0xc4ceb9fe1a85ec53ULL; h ^= h >> 33;
          return h;
      } else {
          // Pointer (string): FNV-1a over string bytes
          LangString* s = (LangString*)(uintptr_t)key;
          uint64_t h = 14695981039346656037ULL;
          for (int64_t i = 0; i < s->length; i++) {
              h ^= (uint8_t)s->data[i]; h *= 1099511628211ULL;
          }
          return h;
      }
  }
  ```
- Add unified equality:
  ```c
  static int lang_ht_key_equal(int64_t a, int64_t b) {
      if ((a & 1) != (b & 1)) return 0;  // different types
      if (a & 1) return a == b;           // both tagged ints
      // Both strings
      LangString* sa = (LangString*)(uintptr_t)a;
      LangString* sb = (LangString*)(uintptr_t)b;
      return sa->length == sb->length && memcmp(sa->data, sb->data, sa->length) == 0;
  }
  ```
- Rewrite all 7 hashtable functions to use unified hash/equality
- Remove all 7 `*_str` function implementations
- Update `lang_for_in_hashtable` — keys are stored as-is (tagged or pointer), pass directly to closure
- Update `lang_hashtable_trygetvalue` — keys stored as-is
- Keep `lang_index_get_str` and `lang_index_set_str` as thin redirects to unified functions
- Update `lang_hashtable_create_str` to just call `lang_hashtable_create`

IMPORTANT: Keys are now stored AS-IS in the hashtable. The compiler no longer untags before set. Integer keys stored tagged (LSB=1), string keys stored as raw pointers (LSB=0).
